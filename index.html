    Descripción del Proyecto

    El presente modelo entidad-relación fue desarrollado con el objetivo de representar de manera clara y estructurada la información relacionada con patinadores, profesores, torneos y sus respectivas categorías dentro de una organización deportiva.

    Listado de Entidades

    • Patinador
    • Profesor
    • Sede
    • Torneo
    • Compentencia

    Ventajas del Sistema

    • Modelo flexible y escalable.
    • Permite registrar múltiples torneos, competencias y cambios anuales.
    • Útil para análisis, gestión y seguimiento del rendimiento de patinadores.

    Base de Datos

    1) Listado completo por sede, con nombre y apellido de los profesores y cargo

    SELECT s.nombre_sede AS 'Sede', CONCAT(p.nombre_profesor,' ',p.apellido_profesor) AS 'Profesor', c.nombre_cargo AS 'Cargo' FROM sede_profesor sp JOIN sede s ON s.id_sede = sp.sede_id JOIN profesor p ON p.id_profesor = sp.profesor_id JOIN profesor_cargo pc ON pc.profesor_id = p.id_profesor JOIN cargo c ON c.id_cargo = pc.cargo_id GROUP BY s.nombre_sede, p.id_profesor, c.id_cargo ORDER BY s.nombre_sede, p.apellido_profesor;

    2) Listado completo de patinadores indicando en que categoría compitieron los últimos 3 años

    SELECT CONCAT( p.nombre_patinador, ' ', p.apellido_patinador ) AS 'Patinador', r.rango_edad AS 'Rango de edad', e.eficiencia_cat AS 'Nivel de eficiencia', d.division_cat AS 'División', t.anio_torneo AS 'Año del torneo' FROM patinador p JOIN categoria c ON p.id_patinador = c.patinador_id JOIN rango_edad_cat r ON c.rango_edad_id = r.id_rango_edad JOIN division_cat d ON c.division_cat_id = d.id_division_cat JOIN eficiencia_cat e ON c.eficiencia_cat_id = e.id_eficiencia_cat JOIN torneo t ON p.torneo_id = t.id_torneo WHERE t.anio_torneo > YEAR(CURDATE()) - 3 ORDER BY p.apellido_patinador, p.nombre_patinador, t.anio_torneo DESC ;

    3) Listado de patinadores menores a 14 años

    SELECT CONCAT( nombre_patinador, ' ', apellido_patinador ) AS 'Patinador', DATE_FORMAT(nacimiento_patinador, '%d-%m-%Y') AS 'Fecha de nacimiento', CONCAT( TIMESTAMPDIFF( YEAR, nacimiento_patinador, CURDATE()), ' años' ) AS 'Edad' FROM patinador WHERE TIMESTAMPDIFF( YEAR, nacimiento_patinador, CURDATE()) < 14;

    4) Listado de patinadores más eficientes indicando a que sede pertenece

    SELECT CONCAT( p.nombre_patinador, ' ', p.apellido_patinador ) AS 'Patinador', s.nombre_sede AS 'Sede', e.eficiencia_cat AS 'Eficiencia más alta' FROM patinador p JOIN categoria c ON p.id_patinador = c.patinador_id JOIN eficiencia_cat e ON c.eficiencia_cat_id = e.id_eficiencia_cat JOIN sede s ON p.sede_id = s.id_sede WHERE e.eficiencia_cat = 'Nivel 1' ORDER BY p.apellido_patinador, p.nombre_patinador;
